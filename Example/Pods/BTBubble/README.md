

# BTBubble(气泡)



这是Swift语言实现的一款气泡能力。

![BTBubbleGif](https://github.com/intsig171/BTBubble/blob/main/Sample/bubbleSmaple.gif)



您可以下载本项目，查看 **Example**  工程，获取更多的使用示例。



### 如何使用

```
pod 'BTBubble'
```

```
import BTBubble

var bubble = BTBubble()
bubble.show(text: "明亮的月光洒在窗户纸上，好像地上泛起了一层白霜。", from: button)
```
